# 📚 Library Management System

## 🚀 Overview

This is a Java-based Library Management System that allows users to manage books efficiently.

## ✨ Features

* Add Books
* Issue Books
* Return Books
* Filter Books (by Author & Category)
* Exception Handling for invalid operations

## 🛠️ Tech Stack

* Java
* OOP Concepts
* Collections Framework

## 📂 Project Structure

* `model/` → Data classes
* `service/` → Business logic
* `exception/` → Custom exceptions
* `main/` → Entry point

## ▶️ How to Run

1. Compile:
   javac LM.java
2. Run:
   java LM
